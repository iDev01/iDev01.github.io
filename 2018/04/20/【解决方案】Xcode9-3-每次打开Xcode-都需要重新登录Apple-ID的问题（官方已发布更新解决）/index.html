【解决方案】Xcode9.3 每次打开Xcode 都需要重新登录Apple ID的问题（官方已发布更新解决）

2018-04-20

现象

每次打开Xcode9.3，在Preference中Apple ID提示“Your session has expired. Please log in.”

原因

Xcode9.3没有去更新钥匙串中的“Xcode-Token”和“Xcode-AlternateDSID”

解决方案

下载一个低版本的Xcode(我下载的是9.2)去登录，生成这两个key，删掉低版本的Xcode，结束。
