python中的验证码识别库PyTesser的安装教程

2016-11-21

最近刚刚接触这个东西，windows上安装的教程，看有个教程里写着添加环境变量，但是具体怎么添加、修改，就比较混乱了。我把安装的过程记录一下。

###需要安装的：

####1.PIL
Windows平台可以直接去PIL官网下载32位的exe安装包。
http://pythonware.com/products/pil/
64位的请去下面这个地址找：
http://www.lfd.uci.edu/~gohlke/pythonlibs/#pillow

####2.下个pytesser和tesseract-ocr的包
我在这里直接提供一个可用的：
链接: https://pan.baidu.com/s/1nvp8PgH 密码: jspm

###安装过程：

#####1.安装PIL
: 32位的exe文件，你应该会装的
: 64位是whl文件，用 pip install xxx.whl 命令就可以了

#####2.安装图片处理模块
: a.将从网盘上下载的压缩包解压到路径/Lib/site-pakeages中。
我的是这个C:/Python27/Lib/site-packages


: b.在site-packages目录中建一个文件pytesser.pth，用文本编辑器打开，写入内容pytesser，然后保存。

: c.进入pytesser文件夹，更改py文件pytesser文件重命名为 __init__.py
打开该文件，修改内容并保存：

: d.接下来就是环境变量了
右键计算机，选择属性，然后按下图中的步骤操作

Path中添加C:\Python27\Lib\site-packages\pytesser\;

现在你可以在你的py文件中通过from pytesser import *引用啦~
